Изучаем REST: Руководство по RESTful

Разработка restful веб-сервисов, подобно другим навыкам в программировании, ничто иное, как частью искусства, частью науки. По мере эволюционирования отрасли веб-разработки, разработка REST API становится более детерминированной, выделяются лучшие практики. Несмотря на то, что нет предписанных стандартов разработки RESTful веб-сервисов, за исключением HTTP, важно спроектировать свой RESTful API в соответствии с передовой практикой для облегчения разработки самим и упрощения использования API клиентом.

В настоящее время существует не так много руководств по REST API для помощи одиноким разработчикам. RestApiTutorial.ru предназначен для отслеживания лучших практик REST API и их агрегирование, для обеспечения быстрого поиска и самообразования. Мы рассмотрим теоретические и практические аспекты создания REST веб-сервисов.

Перейдите к уроку Что такое REST? для рассмотрения концепций и ограничений RESTful архитектуры.
